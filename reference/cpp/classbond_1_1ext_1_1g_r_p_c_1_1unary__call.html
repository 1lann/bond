Detailed Description

template<typename TRequest, typename TResponse>
class bond::ext::gRPC::unary_call< TRequest, TResponse >

The details of a single async, unary call.

Call Finish or FinishWithError to send a response back to the client.

If no explicit call to Finish/FinishWithError has been made before this call is destroyed, a generic internal server error is sent.

Note
This class can only be moved.
