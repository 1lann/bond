Detailed Description

template<typename TRequest, typename TResponse>
struct bond::ext::gRPC::detail::unary_call_impl< TRequest, TResponse >

Implementation class that holds the state associated with a single async, unary call.

It manages its own lifetime: to send a response, it enques itself in the completion queue. When that sending of the response is done and it is dequeued from the completion queue, it deletes itself.
